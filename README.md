This repository tracks local preservation and compatibility work for getting the stock `Laghaim` client to talk to a local emulator.

Current focus:

- Milestone 1: launcher compatibility
- Local server discovery
- Stubbed login success path
- Packet capture and protocol notes

Repository notes:

- Research notes and design docs are committed so progress is easy to track between runs.
